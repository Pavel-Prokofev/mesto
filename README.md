# Проект: Место
***
_Проект личной галереи путешественника, помогающий сохранить воспоминания о интересных местах._

## __Технологии использованные в проекте:__

* flex
* gap
* grid layout
* псевдоэлементы
* transition
* Позиционирование
* Адаптивная вёрстка
* БЭМ Nested
* Figma
* media запросы
* JS
